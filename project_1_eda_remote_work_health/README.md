# Project 1: Exploratory Data Analysis (EDA)

## Dataset
Post-Pandemic Remote Work & Health Impact (2025)

## Objective
To explore and analyze how remote work affects health, wellbeing, and productivity indicators.

## Files
- `post_pandemic_remote_work_health_impact_2025.csv` — cleaned dataset
- `remote_work_health_eda_1_done.ipynb` — EDA notebook with analysis and visualizations

## Analysis Steps
- Data cleaning and preprocessing
- Handling missing values
- Exploratory data analysis (EDA)
- Visualization of key relationships

## Key Insights
- Remote work intensity shows a relationship with stress and wellbeing
- Sleep quality is linked to productivity outcomes
- Work-life balance differs across demographic groups

## Tools
- Python
- Pandas
- Matplotlib
- Seaborn
